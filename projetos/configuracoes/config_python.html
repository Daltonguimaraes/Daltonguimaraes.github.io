⚙️ Configuração de Ambiente Python

Siga os passos abaixo para configurar um ambiente de desenvolvimento com Python e Flask.

  1. Criar o ambiente virtual:
    python3 -m venv ./env
  2. Ativar o ambiente virtual:
    source ./env/bin/activate
  3. Instalar o framework Flask:
    pip install Flask
  4. Verificar a versão do Flask:
    python -m flask --version
  5. Salvar os pacotes instalados:
    pip freeze > requirements.txt
  6. Executar a aplicação:
    python app.py
✅ Certifique-se de estar na raiz do seu projeto ao executar esses comandos.
