⚙️ Guia de Configuração de Ambiente Python

Siga os passos abaixo para configurar um ambiente de desenvolvimento com Python e Flask.

Para instalar o Python 3.10 via linha de comando, o processo varia dependendo do seu sistema operacional. Aqui estão as instruções para Windows, Linux (Ubuntu/Debian e derivados) e macOS:

1. No Windows:

Opção 1: Instalador oficial (recomendado para usuários comuns)

Baixe o instalador do Python 3.10 e execute via linha de comando:

curl -o python-3.10.exe https://www.python.org/ftp/python/3.10.0/python-3.10.0-amd64.exe
    python-3.10.exe /quiet InstallAllUsers=1 PrependPath=1
  • /quiet: Instala silenciosamente.
  • InstallAllUsers=1: Instala para todos os usuários.
  • PrependPath=1: Adiciona Python ao PATH.

Opção 2: Usando Chocolatey (gerenciador de pacotes)

Se você tem o Chocolatey instalado:

choco install python --version=3.10.0

2. No Linux (Ubuntu/Debian e derivados):

Via repositório oficial (método recomendado)

sudo apt update
    sudo apt install software-properties-common -y
    sudo add-apt-repository ppa:deadsnakes/ppa -y  # Repositório com versões mais recentes
    sudo apt update
    sudo apt install python3.10 -y

Verificar instalação:

python3.10 --version

Instalar pip (se necessário):

curl -sS https://bootstrap.pypa.io/get-pip.py | python3.10

3. No macOS:

Via Homebrew (recomendado):

brew update
    brew install python@3.10

Verificar instalação:

python3.10 --version

Alternativa: Usando pyenv (para múltiplas versões do Python)

brew install pyenv  # Instala o pyenv
    pyenv install 3.10.0  # Instala o Python 3.10
    pyenv global 3.10.0  # Define como versão padrão

Pós-instalação (todos os sistemas):

  • Verifique se o Python 3.10 está corretamente instalado:
    python3.10 --version
  • Atualize o pip (gerenciador de pacotes):
    python3.10 -m pip install --upgrade pip

Observações:

  • No Linux, se você precisar alternar entre versões do Python, use update-alternatives ou pyenv.
  • No Windows, certifique-se de marcar "Add Python to PATH" durante a instalação manual (caso não use o método /quiet).
